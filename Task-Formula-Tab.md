#  Task 1: Scholarship Eligibility.

##  Create Table

| A       | B           | C     | D         |
| ------- | ----------- | ----- | --------- |
| Student | Attendance% | Marks | Eligible? |
| Aman    | 85          | 78    |           |
| Simran  | 65          | 82    |           |
| Ravi    | 90          | 88    |           |
| Neha    | 75          | 45    |           |

A student is eligible for scholarship only if:

* Attendance ≥ 75
* Marks ≥ 60

## 🎯 Sample Output

| Student | Attendance% | Marks | Eligible? |
| ------- | ----------- | ----- | --------- |
| Aman    | 85          | 78    | YES       |
| Simran  | 65          | 82    | NO        |
| Ravi    | 90          | 88    | YES       |
| Neha    | 75          | 45    | NO        |
