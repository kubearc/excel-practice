## Task: Employee Salary Report Formatting

Create the following table in Excel and perform all tasks using **Home Tab** and **Page Layout Tab** features.

| Employee ID | Employee Name | Department | Basic Salary | Bonus | Total Salary |
| ----------- | ------------- | ---------- | ------------ | ----- | ------------ |
| EMP101      | Aman          | HR         | 25000        | 3000  |              |
| EMP102      | Simran        | IT         | 32000        | 5000  |              |
| EMP103      | Rahul         | Sales      | 28000        | 4000  |              |
| EMP104      | Neha          | Marketing  | 35000        | 6000  |              |
| EMP105      | Arjun         | Finance    | 30000        | 4500  |              |

---

# Part 1 — Home Tab Tasks

### Font Group

1. Make headings **Bold**.
2. Change heading font size to **14**.
3. Change heading font color to **White**.
4. Apply **Blue Fill Color** to heading row.

### Alignment Group

5. Center align all headings.
6. Apply **Wrap Text** on “Department” heading.
7. Merge & Center cells A1:F1 and write:
   **Employee Salary Report**

### Number Group

8. Format Salary and Bonus columns in **Currency Format**.
9. Display salary values with **comma separator**.

### Styles Group

10. Apply any **Cell Style** to the heading row.
11. Use **Conditional Formatting**:

* Highlight salaries greater than 30000.

### Cells Group

12. Insert one new row below EMP103.
13. Delete the Bonus value of EMP105.

### Editing Group

14. Use **AutoSum** to calculate Total Salary.
15. Use **Find & Replace**:

* Replace “HR” with “Human Resource”.

---

# Part 2 — Page Layout Tab Tasks

### Themes Group

16. Apply any workbook **Theme**.

### Page Setup Group

17. Change page orientation to **Landscape**.
18. Set paper size to **A4**.
19. Add custom margins:

* Top: 1 inch
* Bottom: 1 inch

### Scale to Fit Group

20. Fit worksheet on **1 page width**.

### Sheet Options Group

21. Enable:

* Print Gridlines
* Print Headings

### Arrange Group

22. Insert any shape (example: company logo shape) and align it properly.

---

# Advanced Challenge

23. Add a footer using Page Layout:

* Footer text:
  **Prepared By: Your Name**

24. Add page background color or image.

25. Rename worksheet as:
    **Salary_Report**

After completing this task, you will learn:

* Text formatting

* Conditional formatting
* Page setup for printing
* Worksheet presentation skills
* Professional report design in Excel
